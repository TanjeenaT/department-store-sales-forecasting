# department-store-sales-forecasting
Comprehensive time series analysis and forecasting of Australian department store sales using SARIMA models implemented in R.

## Project Description

This project focuses on forecasting Australian department store sales using time series analysis. The objective is to analyze historical sales data, identify trends and seasonality, and generate future sales forecasts using SARIMA models implemented in R.
The project follows an end-to-end analytical approach, including data exploration, statistical modeling, and forecasting.

Folder Structure  
─ data/ contains raw sales data  
─ script/ contains R scripts for analysis and modeling  
─ report/ contains project report (PDF)

## Technologies Used
- R
- Data Visualization
- Statistical Modeling
- Time Series Analysis
- SARIMA (Seasonal ARIMA)

## Methodology

- Data preprocessing and exploration
- Trend and seasonality analysis
- Stationarity testing
- SARIMA model selection and tuning
- Forecast generation and evaluation

## Results

- Identified strong seasonal patterns in department store sales
- Built SARIMA models suitable for retail sales forecasting
- Generated future sales forecasts with clear visualizations
- Detailed analysis, results, and discussion are available in the project report.

## Project Report

A comprehensive report is included covering:
- Problem formulation
- Literature review
- Methodology
- Results and discussion
- Conclusion and future scope
